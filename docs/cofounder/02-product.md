# 02 — Product

> 🛠 **TL;DR** — One native desktop app. One core loop: launch session → work → earn XP → see skills grow → Promethee remembers. The Discord friction problem is solved by making tracking silent and native.
> 

### The Friction Problem — Solved

> 🟢 Current MVP: open Discord → find room → join → sit passively while tracked. This works but excludes anyone who doesn't use Discord, and breaks flow before the session starts. The native app removes this. You launch Promethee. It tracks silently. You work however you want. Discord stays available — it's no longer a prerequisite.
> 

### V1 — The Only Thing That Ships ✅ decided

| # | Feature | Description |
| --- | --- | --- |
| 01 | Today (single view) | One screen. Quest of the day, active session, habits.  |
| 02 | Focus + Silent XP | Timer starts. Prométhée tracks in background. XP at session end. Optional site blocker. |
| 03 | Skill Tree (auto) | Moves from session and what Promethee “see” of you — never manual. Rigueur, Volonté, Courage. No user action required. |
| 04 | Quests | User-defined long/mid/short term objectives. Daily habit quests, auto-reset. Completion = XP. |
| 05 | Mentor Signal V1 | One message per day. Short, contextual, from yesterday's data. Not a classical chat. A signal. Low/Med/High intensity. |
| 06 | Weekly Report | 3 insights → 3 behaviour changes. From tags and sessions. No vanity stats. |
| 07 | HUD Overlay | Translucent layer over the OS. Timer, quest, XP visible. CMD+Space hides all.  |
| 08 | Onboarding 60s | Name → first quest → first session → first XP. Done. |

### The 90-Day Memory Reveal 🟡 planned — month 3

> 🟢 From day 1, Prométhée silently builds a structured memory profile. At day 90, it surfaces what it knows: *"Here's what I've observed about how you work over the last 3 months."* This is the first planned WOW moment in the product — and the strongest word-of-mouth trigger. It only works if data collection is invisible from session 1.
>
