# 03 — AI Mentor & Memory

> 🧠 **TL;DR** — The moat is not the AI model. It's the persistent, structured memory layer. Every feature is a data collection mechanism. The longer the user stays, the more irreplaceable Promethee becomes.
> 

### What the AI Actually Is

> 🔴 **Emotional intelligence is not a product positioning. It is a training direction.** What we are building concretely: a persistent, structured, evolving memory per user. We do not claim "emotionally intelligent AI." We claim: an AI that knows you, because it has been watching you work for months.
> 

### The Four Memory Layers

- **Profile Layer** — name, objectives, active quests. If the user's main quest is "pass the bar exam," every interaction prioritises that.
- **Behavioural Layer** — all sessions, tags, focus scores, time of day, quests, streaks, XP evolution. Built silently, zero user effort.
- **Emotional Layer** — journal entries, self-reported states, recurring patterns. Not displayed until meaningful.
- **Contextual Layer** — tags linked to quests, future API connections. Cross-references behavioural + emotional data for non-obvious insights.

### The Memory Flywheel

User works → data captured → memory updated → next signal more accurate → user trusts Prométhée more → user works more → richer memory. Each cycle makes the product harder to leave.

### V1 AI — Deliberately Limited

> 🟢 V1 Prométhée is not a simple chat. One message per day, short, contextual, based on yesterday's session data. Low surface area — high trust.
